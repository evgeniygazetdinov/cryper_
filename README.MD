A lightweight, efficient C implementation of a Caesar cipher decoder with text processing capabilities and planned PDF support.<br>

## Features
### Currently Implemented
Caesar Cipher Decryption - Shift-based character transformation<br>

Full Alphabet Support - Handles both uppercase (A-Z) and lowercase (a-z)<br>

Character Preservation - Maintains numbers, symbols, and whitespace<br>

Memory Safe - Proper buffer handling and boundary checks<br>

Cross-Platform - Standard C11 for maximum compatibility<br>


## In Progress
Code optimization and warning resolution<br>

Enhanced error handling<br>

Performance improvements<br>

## Planned Features
PDF Text Extraction - Read encrypted text from PDF documents<br>

File I/O Support - Process .txt, .doc, and other text formats<br>

Frequency Analysis - Automatic shift detection<br>
